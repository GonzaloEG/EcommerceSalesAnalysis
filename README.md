# EcommerceSalesAnalysis
Análisis de Ventas de un dataset de Ecommerce con SQL y PowerBI
"El presente análisis evaluó el desempeño comercial de un ecommerce durante 12 meses (abril 2018 - marzo 2019), enfocándose en ventas, rentabilidad y cumplimiento de objetivos por categoría de producto, esto fue logrado con lenguaje de programación SQL y la herramienta visual PowerBI".
